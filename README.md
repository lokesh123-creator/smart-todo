# Smart Todo App

A simple Smart Todo application built using HTML, CSS, and JavaScript.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as Done / Undo
* Prevent duplicate tasks
* Task counter (Total, Completed, Remaining)
* Press Enter to add tasks
* Clear all tasks with confirmation
* Save tasks using Local Storage
* Automatically reload tasks after page refresh

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
